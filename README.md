# MTM6201 Final

## Overview

Pawsitive Futures is a responsive pet adoption website designed to connect users with animals in need of a home. The website includes three main pages: Home, About, and Contact. It focuses on creating a clean, modern, and accessible user experience while promoting animal rescue and adoption.

---

## Process

I began this project by creating wireframes and mockups in Figma to plan the layout and user flow of the website. These designs helped guide the structure and visual direction of each page.

After finalizing the design, I developed the website using HTML, CSS, and the Bootstrap framework. I used Bootstrap’s grid system and components such as the navbar, cards, forms, and modal to build a consistent layout across all pages.

For the homepage, I created a hero section with a search feature and an adoptable pets gallery using responsive cards and images. On the About page, I focused on storytelling by presenting the organization’s mission, vision, and team. The Contact page includes a structured and validated form for user interaction.

I also incorporated responsive images using the `<picture>` element and `srcset` to improve performance across different devices.

---

## Challenges & Solutions

One of the main challenges I faced was ensuring the website was fully responsive. Some layout elements, especially the navigation and image grid, did not display correctly on smaller screens. I resolved this by using Bootstrap’s responsive classes such as `col-md`, `col-lg`, and `d-none d-md-flex`, and by testing the layout using browser developer tools.

Another challenge was implementing accessibility features. I ensured the use of semantic HTML elements such as `<header>`, `<main>`, and `<footer>`, added alt attributes to images, and included a "Skip to main content" link for better keyboard navigation.

I also worked on improving user interaction by adding form validation and a modal login component using Bootstrap’s built-in functionality.

---

## What I Learned

Through this project, I learned how to effectively use Bootstrap to build responsive and structured layouts. I gained a better understanding of accessibility best practices, including semantic markup, ARIA attributes, and keyboard navigation.

I also learned how to optimize images for performance using responsive techniques like the `<picture>` element and `srcset`.

Additionally, I improved my ability to translate design concepts from Figma into functional web pages and gained experience debugging layout and responsiveness issues.
